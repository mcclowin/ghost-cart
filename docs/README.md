## Current Product State

GhostCart currently supports:

- natural-language product search
- ranked comparison across marketplace and web results
- payment session creation through Stripe or Locus
- payment receipts
- background merchant checkout preparation after payment succeeds

GhostCart does not yet place the final merchant order automatically.
